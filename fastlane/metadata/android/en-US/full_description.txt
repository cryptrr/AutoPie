<p><strong>AutoPie</strong> is a command automation hub for Android. It gives you a Linux environment where you can create, run, and automate Bash commands or Python scripts without using a terminal.</p>

<p>Use it to build small Android utilities for file conversion, downloads, monitoring, backups, web requests, AI tooling, device workflows, and any other task you would normally solve with Linux packages or scripts.</p>

<p><strong>Highlights</strong></p>

<ul>
	<li><strong>Embedded Termux environment</strong>
	<ul>
		<li>Run Linux commands directly from AutoPie without the Terminal</li>
		<li>Install extra Termux packages with <code>pkg install package-name</code>.</li>
		<li>Install Python packages with <code>pip install package-name</code>.</li>
		<li>Includes useful tooling such as Python, pip, openssh and other utilities out of the box.</li>
	</ul>
	</li>
	<li><strong>Command and script automation</strong>
	<ul>
	    <li>Create custom applets that lives on your homescreen.</li>
		<li>Create applets from Bash commands and Python scripts.</li>
		<li>Add commands to the Android share sheet for quick file, text, and URL workflows.</li>
		<li>Watch folders and automatically process new files.</li>
		<li>Schedule commands with cron-style automation.</li>
		<li>Use interactive command mode when a script needs a live shell session.</li>
	</ul>
	</li>
	<li><strong>Custom applet creation forms</strong>
	<ul>
		<li>Add text inputs, switches, selectors, file inputs, and other extras to your commands.</li>
		<li>Turn scripts into simple Android workflows that are easy to reuse.</li>
		<li>Use placeholders for files, URLs, text, filenames, directories, random values, and more.</li>
	</ul>
	</li>
	<li><strong>Ready for practical phone automation</strong>
	<ul>
		<li>Convert videos, audio, images, screenshots, GIFs, and documents with command-line tools.</li>
		<li>Upload, download, rename, compress, and organize files automatically.</li>
		<li>Run web checks, API calls, health monitors, notifications, and custom background jobs.</li>
		<li>Keep the command output and logs accessible from the app.</li>
	</ul>
	</li>
	<li><strong>Open source Android app</strong>
	<ul>
		<li>Built with Kotlin, Jetpack Compose, and Material 3.</li>
		<li>Designed for users who want powerful local automation without ads or clutter.</li>
	</ul>
	</li>
</ul>
